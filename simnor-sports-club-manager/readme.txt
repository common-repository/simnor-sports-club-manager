Simnor Sports Club Manager

== Description ==

Simnor Sports Club Manager allows you to easily manage a sports team fixtures, results and players.

Shortcodes available:

[fixtures_table season="2013 - 2014" number="10" team="First Team" what_to_show="results"]

Variables:
"season" is the name of the season that you want to show
"team" is the name of the team that you want to show
"number" is the number of fixtures you'd like to show
"what_to_show" is the type of fixtures you want to show, leave blank for all, "results" for past fixtures and "fixtures" for future games.

[players team="Team Name"]

Variables: team is the name of the team you'd like to show.

== Installation ==

1. Upload the `simnor-sports-club-manager` folder to your to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
